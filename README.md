# Công Cụ Của Tôi

Trang web công cụ tải video Facebook, TikTok miễn phí

## Tính năng
- Tải video Facebook
- Tải video TikTok
- Tách nhạc TikTok
- Lưu bài viết Facebook

## Công nghệ
- Node.js
- HTML/CSS/JS
- Vercel

## Cài đặt
\`\`\`bash
git clone https://github.com/ncd405/Cong-Cu-Cua-Toi.git
cd Cong-Cu-Cua-Toi
npm install
npm start
\`\`\`
